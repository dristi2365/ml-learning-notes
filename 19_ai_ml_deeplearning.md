# AI vs Machine Learning vs Deep Learning

Think of them as nested circles:
AI (biggest) → ML (subset of AI) → Deep Learning (subset of ML)

## Artificial Intelligence (AI)
The broadest concept. Any technique that enables machines 
to mimic human intelligence — thinking, reasoning, problem 
solving, understanding language, recognizing images.

Examples: Chess playing programs, spam filters, Siri, 
self driving cars, recommendation systems.

## Machine Learning (ML)
A subset of AI. Instead of programming explicit rules, 
the machine learns patterns from data on its own.

Example: Instead of writing rules like "if email contains 
free money it is spam", you show the model thousands of 
spam emails and it figures out the pattern itself.

## Deep Learning (DL)
A subset of ML. Uses neural networks with many layers 
to learn complex patterns. Especially powerful for 
images, audio and text.

Examples: Face recognition, voice assistants, 
ChatGPT, YOLO.

## Simple analogy:
- AI = the goal (make machines smart)
- ML = one way to achieve that goal (learn from data)
- Deep Learning = a powerful technique within ML 
  (using neural networks with many layers)

## Key differences:
| | AI | ML | Deep Learning |
|--|--|--|--|
| Scope | Broadest | Subset of AI | Subset of ML |
| Approach | Any technique | Learning from data | Neural networks |
| Data needed | Varies | Moderate | Large amounts |
| Examples | Chess AI | Spam filter | ChatGPT, YOLO |

## Simple rule to remember:
All Deep Learning is ML.
All ML is AI.
But not all AI is ML.
And not all ML is Deep Learning.
