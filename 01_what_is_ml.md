# What is Machine Learning?

In normal programming, you write exact rules yourself.
Example: if temperature > 30: print("It's hot")

In Machine Learning, instead of writing rules yourself, you show the computer lots of examples and let it figure out the rules on its own.

**Simple definition:**
Machine Learning is when a computer learns patterns from data instead of following rules you wrote yourself.

**Real example:**
YOLO is a machine learning model. Nobody wrote rules like "if these pixels look like this, it's a person." Instead it was shown millions of images of people and learned what 
a person looks like on its own.
