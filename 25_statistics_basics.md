# Mean, Median, Mode and Standard Deviation

## Mean (Average)
Add all values, divide by count.
Sensitive to outliers - one extreme value can pull it far.

## Median (Middle value)
Sort data, pick the middle value.
Not affected by outliers - more reliable when extreme values exist.

## Mode (Most frequent value)
The value that appears most often.
Useful for categorical data.

## Standard Deviation (Spread)
Measures how spread out values are from the mean.
- Low std dev = consistent values
- High std dev = inconsistent values
Two datasets can have the same mean but very different std dev.

## Quick summary:
| Measure | Tells you | Affected by outliers? |
|---------|-----------|----------------------|
| Mean | Average | Yes |
| Median | Middle value | No |
| Mode | Most common | No |
| Std Dev | Spread | Yes |
