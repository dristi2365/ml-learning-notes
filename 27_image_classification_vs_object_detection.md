# Image Classification vs Object Detection

## Image Classification
The model tells you WHAT is in an image - one label 
for the whole image.
Example: "This image contains a cat."

## Object Detection
The model tells you WHAT is in an image AND WHERE - 
draws bounding boxes around each detected object.
Example: "There is a cat at coordinates (120, 340) 
and a dog at coordinates (450, 200)."

## Simple analogy:
- Image classification = looking at a photo and saying 
  "this is a family photo"
- Object detection = pointing at each person and the dog 
  individually and saying where they are
